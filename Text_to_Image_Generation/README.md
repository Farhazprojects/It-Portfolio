#  Imagination to Image — Copilot Style

An interactive AI app that transforms text prompts into vivid images using Stable Diffusion and Gradio.  
Users type their imagination, adjust settings, and instantly see results in a clean Copilot‑style interface.

##  Features
- **Copilot‑style UI**: Simple, guided interface with textboxes and sliders.
- **Exact prompt fidelity**: Whatever the user types is passed directly to the model.
- **Negative prompts**: Refine outputs by excluding unwanted details.
- **CPU‑friendly**: Runs on Hugging Face Spaces free tier (no GPU required).
- **Colab‑ready**: Works seamlessly in Google Colab for learners with low‑spec PCs.
